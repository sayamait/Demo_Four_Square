For the Current Basic Implementation the Data used: Yelp Dataset Challenge & realtime traffic data fetched from Google-Map API.

Yelp Dataset:

1.6M reviews and 500K tips by 366K users for 61K businesses.

481K business attributes, e.g., hours, parking availability, ambience.

Social network of 366K users for a total of 2.9M social edges.

Aggregated check-ins over time for each of the 61K businesses.

US Cities available in the data:

Pittsburgh, Charlotte, Urbana-Champaign, Phoenix, Las Vegas, Madison.

Input & Output of the current implementation:

For the current app implementation the user should enter the current physical address & the radius (in Miles) to search .

The current app implementation will give a Pandas table output containing: 1) Average historic chekin in the current hour & day of the week; 2) Name of the place; 3) Rating of the place; 4) Travelling Time .

The first three output columns are generated using the Yelp Dataset & the last one using Google API.

Please enter the distance(in mile) to perform search: for example:15 to search within 15 mile radius
Please enter the current address: for example:153 Mountain Dr Carnegie, PA 15106

NB:There are inconsistencies in the current version. For example, the sequential search is slow for production purpose, it might not work with any arbitrary address currently, as more analysis & debugging is required but the example entry address is tested. Currently working on Improving.
